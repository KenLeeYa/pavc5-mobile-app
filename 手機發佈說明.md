# 新版實用視聽華語 5 手機版發佈說明

這個資料夾是一個可發佈的 PWA。學生用手機打開網址後，可以把它加到主畫面，像 App 一樣使用。

## 最簡單的分享方式

1. 將整個資料夾上傳到支援 HTTPS 的靜態網站服務，例如 Netlify、Vercel、GitHub Pages 或學校網站空間。
2. 把產生的網址傳給學生。
3. 學生用手機瀏覽器打開網址。
4. Android Chrome：點右上角選單，選「加到主畫面」或「安裝應用程式」。
5. iPhone Safari：點分享按鈕，選「加入主畫面」。

## 課堂臨時測試方式

老師的電腦和學生手機連同一個 Wi-Fi，老師在資料夾內執行：

```powershell
python -m http.server 4174 --bind 0.0.0.0
```

再把電腦的區網網址給學生，例如：

```text
http://192.168.1.104:4174/
```

這種方式只適合課堂臨時測試。正式分享建議使用 HTTPS 網站，手機才更容易安裝與離線快取。

## 匯入教材內容

App 不內建課本全文、完整生詞表或音檔。請把合法擁有或自行整理的內容填入 `content-template.json` 格式，再從 App 的「匯入」頁載入。

資料格式重點：

- `cards`：生詞、成語、俗語、四字詞。說明與例句都要有拼音與越文翻譯。
- `grammar`：語法說明。說明與例句都要有拼音與越文翻譯。
- `exercises`：每課練習題。只放中文題目、中文選項、中文答案、指定生詞與中文解析，不放拼音或越文翻譯。
- 造句題使用 `type: "sentence"` 和 `requiredTerms`，系統會檢查學生句子是否包含指定生詞與語法關鍵詞。
