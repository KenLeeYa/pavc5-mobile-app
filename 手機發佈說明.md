# 新版實用視聽華語 5 手機版發佈說明

這個資料夾是一個可發佈的 PWA。學生用手機打開網址後，可以把它加到主畫面，像 App 一樣使用。

## 最簡單的分享方式

1. 將整個資料夾上傳到支援 HTTPS 的靜態網站服務，例如 Netlify、Vercel、GitHub Pages 或學校網站空間。
2. 把產生的網址傳給學生。
3. 學生用手機瀏覽器打開網址。
4. Android Chrome：點右上角選單，選「加到主畫面」或「安裝應用程式」。
5. iPhone Safari：點分享按鈕，選「加入主畫面」。

## 課堂臨時測試方式

老師的電腦和學生手機連同一個 Wi-Fi，老師在資料夾內執行：

```powershell
python -m http.server 4174 --bind 0.0.0.0
```

再把電腦的區網網址給學生，例如：

```text
http://192.168.1.104:4174/
```

這種方式只適合課堂臨時測試。正式分享建議使用 HTTPS 網站，手機才更容易安裝與離線快取。

## 手機瀏覽器暫存問題

這個 App 使用 PWA 與 Service Worker，手機可能會暫時保留舊版檔案。新版已改成「網路優先、失敗才用快取」，並在偵測到新版時顯示「重新整理」提示。

如果學生仍然看到舊畫面，請依序處理：

1. 在 App 畫面按「重新整理」提示，或關掉再重新打開。
2. Android Chrome：網址列旁選單 → 網站設定 → 清除資料，然後重開網址。
3. iPhone Safari：設定 → Safari → 進階 → 網站資料，搜尋網站網域後刪除資料。
4. 若是加入主畫面的 App，先刪除主畫面圖示，再用 Safari/Chrome 打開最新網址重新加入。

每次程式更新後，`sw.js` 的快取版本也要同步增加；目前版本為 `pavc5-mobile-v4-20260701`。

## 匯入教材內容

App 目前內建你提供照片整理的第一課內容。其他課請把合法擁有或自行整理的內容填入 `content-template.json` 格式，再從 App 的「匯入」頁載入。

資料格式重點：

- `texts`：課文。每句可含中文、拼音、越文翻譯，並用 `marks` 標註生詞、語法、專有名詞、成語/俗語/四字詞。
- `cards`：生詞、成語、俗語、四字詞。說明與例句都要有拼音與越文翻譯。
- `grammar`：語法說明。說明與例句都要有拼音與越文翻譯，也可加 `practice` 顯示語法練習與遮蔽答案。
- `exercises`：每課練習題。只放中文題目、中文選項、中文答案、指定生詞與中文解析，不放拼音或越文翻譯。
- 造句題使用 `type: "sentence"` 和 `requiredTerms`，系統會檢查學生句子是否包含指定生詞與語法關鍵詞。
