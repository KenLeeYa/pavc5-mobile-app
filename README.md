# 華語 5 隨身學

這是一個給越南學生使用的《實用視聽華語 5》手機學習 PWA 原型。

## 已有功能

- 手機優先介面，可在 Chrome/Edge 加到主畫面。
- 第三版第五冊 14 課課程骨架。
- 字詞卡：繁體字、拼音、中文說明、說明拼音、越南語解釋、中文例句、例句拼音、越文翻譯。
- 語法卡：語法形式、語法拼音、中文說明、說明拼音、越南語說明、中文例句、例句拼音、越文翻譯。
- 每課練習題：包含生詞理解、語法選擇、依照生詞與語法造句；只顯示中文題目與中文選項，不顯示拼音或越文；作答後自動訂正並揭示答案或參考答案。
- 瀏覽器中文語音朗讀。
- 三題快測，立即顯示對錯。
- localStorage 儲存學習進度。
- JSON 匯入自己的合法教材內容。
- Service Worker 離線快取。

## 版權處理

《實用視聽華語》是受版權保護的教材。本專案不收錄課本全文、完整生詞表或音檔。

公開資料可確認的教材輪廓：

- 第三版第五冊商品資訊標示 14 課、1004 個生詞、70 則語法。
- 網路上可找到部分課名、教學影片或二手書資訊，但不應直接抓取未授權 PDF/掃描全文放入 App。

建議做法是由老師或學生把自己合法擁有的課本內容整理成 `content-template.json` 格式，再透過 App 的「匯入」頁載入。

## 匯入格式

請參考 `content-template.json`：

```json
{
  "cards": [
    {
      "id": "1-001",
      "lesson": 1,
      "term": "團圓",
      "pinyin": "tuan2 yuan2",
      "meaningVi": "sum họp",
      "example": "春節時，許多家庭會一起吃團圓飯。",
      "audioUrl": ""
    }
  ]
}
```

App 會載入 `cards`、`grammar` 與 `exercises`。練習題請只放中文題目、中文選項、指定生詞、中文答案與中文解析；不要放拼音或越文翻譯。造句題可使用 `type: "sentence"` 與 `requiredTerms`，系統會檢查學生句子是否包含指定生詞與語法關鍵詞。

## 本機預覽

在此資料夾執行：

```powershell
python -m http.server 4173
```

然後打開 `http://localhost:4173/`。
